Pentru a folosi acest site aveti nevoie de XAMPP Control Panel
Puneti in  folderul C:\xampp\htdocs proiectul
In acesta va trebui sa dati start la MySQL si la Apache
Dupa intrati pe un browser si scrieti localhost/phpmyadmin si imporati baza de date cu numele food_db.sql
Pentru a intra pe site scrieti localhost/BSFOOD iar pentu a acesa panoul de admin localhost/BSFOOD/admin/admin_login
